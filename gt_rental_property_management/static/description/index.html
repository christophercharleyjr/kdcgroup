Odoo Property Lease / Rental / Tenancy Management

Property management module in Odoo is built for Property Landlords, Property Brokers and Tenants. This property management module has two entities one is Property another is Tenancy. In this module we configure different properties with their tenancy details, tenancy issue, tenancy agreement, tenancy payment, landlord’s details, landlords payment and related documents..

How the module of Odoo Property management –After installation of this module you can see one main menu as Property Rental Application.

Let’s check how we can configure Different Properties. Now we have to create Property. Click Property menu and hit create button.


Create Property form will open up. Under Property first configure Property name. We can upload photograph of property. Select Landlord from list. If you want to create new landlord then clicks on “Create and Edit” option.

Landlord creation form will open. Add Landlord name and hit on create option.

Once landlord details form will pop-up. Add Landlord details like Landlord’s Address, Phone number, Mobile number, Email, Login and password. Hit save button after adding details. Landlord will get saved and it will add in Landlord pop-up. Again hit on save button.


New created Landlord will add in property form. Fill up all other details of Property form. eg. Property Address, Property Type, Bedrooms, Bathrooms, Gas safety Expiry. If you have file related Gas and Electricity then you can upload that file.

Landlord and property created successfully. Now we have to move on Tenants.
Click on Tenants Details Menu and hit on Create button to create Tenant.

Tenant form will open. Add Tenant name in create Tenant pop-up. Hit Create and Edit button after adding name.

Fill up all details in Tenant form like Address, Email, Login, and Password.

Now add Tenant date of birth, Gender, Occupation in previous Tenant (1st one) form.

Again move on property. Your Property status is ‘Free’ .Now we rent that property to tenant. Click on Create Tenancy Button.

• Select Tenant from list and fill up all details like
• Tenancy Start: add tenancy start date.
• Tenancy End: add end tenancy date.
• Deposit: add deposit amount.
• Payment Day: Payment date that date which Agent get payment from Tenant for every month.
• Landlord Day: Landlord day means that day which Agent give payment to Landlord.

Commission Type: You can select either Fix or Percentage. If you select percentage add your commission in percentage and commission will calculate in commission field.


Now property status will changed into Not available and tenant also gets added.

Tenancy gets add in property..

Here Agent Property creation of Landlord and that property give on rent to Tenant work done.
Explanation of Tenant’s Activity as follows.
After you Login with Tenant details:
Tenant can see his rented properties. Tenant can create issue if tenant has any property issue using Create Issue Button.

Add Property related issue and tenant can upload issue related image also

Tenant can see All issues which he created from Issue Menu..

Tenant see property rental agreement.

Now we are moving on Agent activities. Agent can check Properties related issues and agent can add Handy man and cost for repair that issue.

This cost will charged to landlords. Hit on Request button to send request to landlords about chargeable cost. After requesting Issues status will gets changed into Processing.

Once solve that problem Click on Done button and status will change into Done.

Agent can check Payments of Issue using Issue Payment menu.


Agent charge Repair cost to landlord using Charge Landlord button.



Once pop-up open, add amount and click on Charge Landlord button.

Explanation of Landlord Activities as follows:
Login with Landlord’s login details. Landlord can check all his properties details.

Landlord can check how much amount charged for issue.

Agent can see all landlords due payments and all tenants due payments.
